# Подсказка по GIT

## Создание репозитория
```sh
git init
```
## Добавление файла для перезаписи состояния
```sh
git add
```

## Вывод перечня изменений подробно
```sh
git log
```

## Вывод перечня изменений сокращенно
```sh
git log --oneline
```
## Перемещение по состояниям модификации файла
```sh
git checkout <nuber_state>
```
